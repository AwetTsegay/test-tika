This project is designed to extract, process, and convert various types of embedded files from a given document. It leverages advanced file handling and data manipulation techniques to ensure that all relevant information is captured and made accessible in multiple formats.

## Main Features
- **File Extraction**: Efficiently extracts embedded files from a variety of document types.
- **File Sorting**: Organizes extracted files by their types (e.g., JSON, TXT, PDF) for easier access and management.
- **Data Conversion**: Converts extracted files into different formats, including JSON, Parquet, CSV, and Excel, to meet diverse user needs.
- **Comprehensive Processing**: Handles text, image, and binary files appropriately, ensuring that all content is preserved and accessible.

## Purpose
The primary goal of this project is to provide a robust solution for users who need to work with embedded files within documents. By automating the extraction and conversion process, the project aims to save users time and effort while ensuring data integrity and accessibility.
    
## Usage
Users can utilize this project to streamline their workflow when dealing with documents containing embedded files. The project is particularly useful for data analysts, researchers, and developers who require efficient access to various file types for further analysis or processing.
